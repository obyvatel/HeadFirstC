Такие замечания по Allegro:
1. Версия которую смог запустить: Allegro 5.2.2.
Используется версия x86: allegro-mingw-gcc6.2.0-x86-dynamic-5.2.2.zip
	The .zip contains three directories under allegro\: bin\ include\ lib\.
	Extract the files from each to eg. C:\MinGW\bin, C:\MinGW\include, C:\MinGW\lib respectively.
Использовал MinGW32.
Для более новых версий как будто использовать MinGW-w64 - for 32 and 64 bit Windows.
Не стал пока разбираться с MinGW-w64.
Вот ссылка на разбор проблемы:
https://stackoverflow.com/questions/55042427/undefined-reference-using-allegro-5-with-mingw-w64
2. Описание быстрого старта: https://github.com/liballeg/allegro_wiki/wiki/Quickstart
3. Tutorial: https://github.com/liballeg/allegro_wiki/wiki/Allegro-Vivace

Как определить версию gcc: gcc -v
